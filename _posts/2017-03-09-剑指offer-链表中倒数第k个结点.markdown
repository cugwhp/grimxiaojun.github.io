## 题目描述

输入一个链表，输出该链表中倒数第k个结点。



两个指针，第一个指针指到第k位时，第二个指针开始操作，

因为总长相同，n=k+m  ;  n=m+k;

代码如下：

```c++
class Solution {
public:
    void reOrderArray(vector<int> &array) {
        vector<int> a,b;
        for(int i=0;i<array.size();++i){
            if(array[i]%2!=0){
                a.push_back(array[i]);
            }else{
                b.push_back(array[i]);
            }
        }
        array.clear();
        array.insert(array.end(),a.begin(),a.end());
        array.insert(array.end(),b.begin(),b.end());
    }
};
```
